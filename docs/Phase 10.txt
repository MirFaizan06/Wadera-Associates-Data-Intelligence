## Phase 10: Developer Admin Dashboard (Super Admin)

Note: The code written in this Phase file is only dummy and placeholder. You write your own logic, and especially styles and structure. we are using custom css, not tailwind.

In this phase, we build the Developer Admin dashboard – the super admin panel with full system oversight. This includes:

- **System Health**: real‑time server metrics (CPU, memory, uptime, disk usage).
- **Log Viewer**: tail recent application logs (Winston logs stored in files), with filtering by level (error, warn, info) and search.
- **Error Monitoring**: aggregated error rates, recent errors, and stack traces.
- **Analytics**: user signup trends, purchase trends, top datasets, traffic overview (from request logs).
- **Permission Enforcement**: only users with Developer role (or `ALL` permission) can access.

We'll use the existing Winston logging and add a simple in‑memory or file‑based log viewer. For analytics, we'll query the database for aggregated stats.

### Step 10.1: Update Prisma Schema (if needed)

We might want a `SystemLog` table to store important events, but we can also read directly from log files. For simplicity and to meet the “no external tools” requirement, we'll keep using Winston file logs and provide an API to read them. However, for error rates and trends, we need structured data. We'll create a `SystemLog` model to store errors and warnings (optional) and also store request logs if needed.

Add to `prisma/schema.prisma`:

```prisma
model SystemLog {
  id        String   @id @default(cuid())
  level     String   // "error", "warn", "info"
  message   String
  meta      Json?    // additional data (stack, user, etc.)
  timestamp DateTime @default(now())
  @@index([level])
  @@index([timestamp])
}
```

But to avoid over‑engineering, we can skip this and rely on file logs. However, the dashboard needs to display recent errors and error rates. We could parse the log files on demand, but that's inefficient. So we'll create a `SystemLog` model and modify the error‑handling middleware to log errors to the database as well as to Winston.

We'll also need a mechanism to log request counts (for traffic) – we could use a middleware that records each request in a `RequestLog` table, but that might be too heavy. Instead, we can aggregate counts in memory and flush to a `Stats` table every hour, as per the requirement. For simplicity, we'll just query purchase and user tables for now, and leave advanced traffic monitoring as optional.

Let's implement a simple `SystemLog` model for errors and important events.

```typescript
import { Request, Response } from 'express';
import { PrismaClient } from '@prisma/client';
import os from 'os';
import fs from 'fs';
import path from 'path';
import { exec } from 'child_process';
import util from 'util';

const prisma = new PrismaClient();
const execAsync = util.promisify(exec);

// Helper to read last N lines from a file
async function tailFile(filePath: string, lines: number = 100): Promise<string[]> {
  try {
    if (!fs.existsSync(filePath)) return [];
    const { stdout } = await execAsync(`tail -n ${lines} "${filePath}"`);
    return stdout.split('\n').filter(line => line.trim() !== '');
  } catch (error) {
    console.error('Error tailing file:', error);
    return [];
  }
}

// ========== System Health ==========
export const getSystemHealth = async (req: Request, res: Response) => {
  try {
    const uptime = os.uptime();
    const totalMem = os.totalmem();
    const freeMem = os.freemem();
    const usedMem = totalMem - freeMem;
    const memoryUsage = (usedMem / totalMem) * 100;
    const cpuLoad = os.loadavg(); // 1, 5, 15 minutes
    const diskUsage = await getDiskUsage(); // optional

    res.json({
      success: true,
      data: {
        uptime,
        memory: {
          total: totalMem,
          free: freeMem,
          used: usedMem,
          usagePercent: memoryUsage.toFixed(2),
        },
        cpu: {
          load1: cpuLoad[0],
          load5: cpuLoad[1],
          load15: cpuLoad[2],
          cores: os.cpus().length,
        },
        disk: diskUsage,
        hostname: os.hostname(),
        platform: os.platform(),
        arch: os.arch(),
      },
    });
  } catch (error) {
    console.error(error);
    res.status(500).json({ success: false, error: 'Server error' });
  }
};

async function getDiskUsage() {
  // This is platform-specific. On Linux, we can use `df -k .`
  try {
    const { stdout } = await execAsync('df -k . | tail -1');
    const parts = stdout.trim().split(/\s+/);
    if (parts.length >= 5) {
      const total = parseInt(parts[1]) * 1024; // KB to bytes
      const used = parseInt(parts[2]) * 1024;
      const available = parseInt(parts[3]) * 1024;
      const usePercent = parseInt(parts[4].replace('%', ''));
      return { total, used, available, usePercent };
    }
  } catch (error) {
    console.error('Disk usage error:', error);
  }
  return null;
}

// ========== Logs ==========
export const getLogs = async (req: Request, res: Response) => {
  try {
    const { lines = 100, level } = req.query;
    const logDir = path.join(__dirname, '../../logs'); // adjust to your log directory
    const logFile = level === 'error' ? 'error.log' : 'combined.log';
    const filePath = path.join(logDir, logFile);

    const logs = await tailFile(filePath, parseInt(lines as string));
    res.json({ success: true, data: logs });
  } catch (error) {
    console.error(error);
    res.status(500).json({ success: false, error: 'Server error' });
  }
};

// ========== Error Stats ==========
export const getErrorStats = async (req: Request, res: Response) => {
  try {
    const now = new Date();
    const oneHourAgo = new Date(now.getTime() - 60 * 60 * 1000);
    const oneDayAgo = new Date(now.getTime() - 24 * 60 * 60 * 1000);
    const oneWeekAgo = new Date(now.getTime() - 7 * 24 * 60 * 60 * 1000);

    const [lastHour, lastDay, lastWeek, recentErrors] = await Promise.all([
      prisma.systemLog.count({ where: { level: 'error', timestamp: { gte: oneHourAgo } } }),
      prisma.systemLog.count({ where: { level: 'error', timestamp: { gte: oneDayAgo } } }),
      prisma.systemLog.count({ where: { level: 'error', timestamp: { gte: oneWeekAgo } } }),
      prisma.systemLog.findMany({
        where: { level: 'error' },
        orderBy: { timestamp: 'desc' },
        take: 20,
      }),
    ]);

    res.json({
      success: true,
      data: {
        lastHour,
        lastDay,
        lastWeek,
        recentErrors,
      },
    });
  } catch (error) {
    console.error(error);
    res.status(500).json({ success: false, error: 'Server error' });
  }
};

// ========== Analytics ==========
export const getAnalytics = async (req: Request, res: Response) => {
  try {
    const now = new Date();
    const thirtyDaysAgo = new Date(now.getTime() - 30 * 24 * 60 * 60 * 1000);

    // Signups per day (last 30 days)
    const signups = await prisma.$queryRaw`
      SELECT DATE(createdAt) as date, COUNT(*) as count
      FROM User
      WHERE createdAt >= ${thirtyDaysAgo}
      GROUP BY DATE(createdAt)
      ORDER BY date ASC
    `;

    // Purchases per day (successful)
    const purchases = await prisma.$queryRaw`
      SELECT DATE(purchasedAt) as date, COUNT(*) as count, SUM(amountINR) as revenue
      FROM Purchase
      WHERE status = 'SUCCESS' AND purchasedAt >= ${thirtyDaysAgo}
      GROUP BY DATE(purchasedAt)
      ORDER BY date ASC
    `;

    // Top datasets by purchase count
    const topDatasets = await prisma.$queryRaw`
      SELECT ts.id, ts.name, COUNT(p.id) as purchaseCount, SUM(p.amountINR) as revenue
      FROM TimeSeries ts
      JOIN Purchase p ON p.timeSeriesId = ts.id
      WHERE p.status = 'SUCCESS'
      GROUP BY ts.id, ts.name
      ORDER BY purchaseCount DESC
      LIMIT 10
    `;

    // Total users
    const totalUsers = await prisma.user.count();
    const totalPurchases = await prisma.purchase.count({ where: { status: 'SUCCESS' } });
    const totalRevenue = await prisma.purchase.aggregate({ where: { status: 'SUCCESS' }, _sum: { amountINR: true } });

    res.json({
      success: true,
      data: {
        signups,
        purchases,
        topDatasets,
        totals: {
          users: totalUsers,
          purchases: totalPurchases,
          revenue: totalRevenue._sum.amountINR || 0,
        },
      },
    });
  } catch (error) {
    console.error(error);
    res.status(500).json({ success: false, error: 'Server error' });
  }
};

// ========== System Logs from DB (optional) ==========
export const getSystemLogs = async (req: Request, res: Response) => {
  try {
    const page = parseInt(req.query.page as string) || 1;
    const limit = parseInt(req.query.limit as string) || 50;
    const skip = (page - 1) * limit;
    const level = req.query.level as string;

    const where: any = {};
    if (level) where.level = level;

    const logs = await prisma.systemLog.findMany({
      where,
      orderBy: { timestamp: 'desc' },
      skip,
      take: limit,
    });
    const total = await prisma.systemLog.count({ where });

    res.json({
      success: true,
      data: logs,
      pagination: { page, limit, total, pages: Math.ceil(total / limit) },
    });
  } catch (error) {
    console.error(error);
    res.status(500).json({ success: false, error: 'Server error' });
  }
};
```

#### 10.5.2: SystemHealth Component

`client/src/pages/admin/SystemHealth.tsx`:

```tsx
import { useEffect, useState } from 'react';
import api from '../../services/api';
import toast from 'react-hot-toast';
import { CircularProgressbar, buildStyles } from 'react-circular-progressbar';
import 'react-circular-progressbar/dist/styles.css';

interface HealthData {
  uptime: number;
  memory: { total: number; free: number; used: number; usagePercent: string };
  cpu: { load1: number; load5: number; load15: number; cores: number };
  disk: { total: number; used: number; available: number; usePercent: number } | null;
  hostname: string;
  platform: string;
  arch: string;
}

export default function SystemHealth() {
  const [health, setHealth] = useState<HealthData | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    api.get('/admin/dev/health')
      .then(res => setHealth(res.data.data))
      .catch(() => toast.error('Failed to fetch health'))
      .finally(() => setLoading(false));
  }, []);

  if (loading) return <div>Loading...</div>;
  if (!health) return <div>No data</div>;

  const formatBytes = (bytes: number) => {
    if (bytes === 0) return '0 Bytes';
    const k = 1024;
    const sizes = ['Bytes', 'KB', 'MB', 'GB', 'TB'];
    const i = Math.floor(Math.log(bytes) / Math.log(k));
    return parseFloat((bytes / Math.pow(k, i)).toFixed(2)) + ' ' + sizes[i];
  };

  const formatUptime = (seconds: number) => {
    const days = Math.floor(seconds / 86400);
    const hours = Math.floor((seconds % 86400) / 3600);
    const minutes = Math.floor((seconds % 3600) / 60);
    return `${days}d ${hours}h ${minutes}m`;
  };

  return (
    <div>
      <h1 className="text-2xl font-bold text-[#723E31] mb-6">System Health</h1>
      <div className="grid grid-cols-1 md:grid-cols-2 gap-6">
        <div className="bg-white p-6 rounded shadow">
          <h2 className="text-lg font-semibold mb-4">Host Info</h2>
          <p>Hostname: {health.hostname}</p>
          <p>Platform: {health.platform}</p>
          <p>Arch: {health.arch}</p>
          <p>Uptime: {formatUptime(health.uptime)}</p>
        </div>

        <div className="bg-white p-6 rounded shadow">
          <h2 className="text-lg font-semibold mb-4">Memory</h2>
          <div className="w-32 h-32 mx-auto mb-4">
            <CircularProgressbar
              value={parseFloat(health.memory.usagePercent)}
              text={`${health.memory.usagePercent}%`}
              styles={buildStyles({
                textSize: '16px',
                pathColor: health.memory.usagePercent > 80 ? '#dc2626' : '#10b981',
                textColor: '#723E31',
              })}
            />
          </div>
          <p>Total: {formatBytes(health.memory.total)}</p>
          <p>Used: {formatBytes(health.memory.used)}</p>
          <p>Free: {formatBytes(health.memory.free)}</p>
        </div>

        <div className="bg-white p-6 rounded shadow">
          <h2 className="text-lg font-semibold mb-4">CPU</h2>
          <p>Cores: {health.cpu.cores}</p>
          <p>Load 1m: {health.cpu.load1.toFixed(2)}</p>
          <p>Load 5m: {health.cpu.load5.toFixed(2)}</p>
          <p>Load 15m: {health.cpu.load15.toFixed(2)}</p>
        </div>

        {health.disk && (
          <div className="bg-white p-6 rounded shadow">
            <h2 className="text-lg font-semibold mb-4">Disk Usage</h2>
            <div className="w-32 h-32 mx-auto mb-4">
              <CircularProgressbar
                value={health.disk.usePercent}
                text={`${health.disk.usePercent}%`}
                styles={buildStyles({
                  textSize: '16px',
                  pathColor: health.disk.usePercent > 80 ? '#dc2626' : '#10b981',
                  textColor: '#723E31',
                })}
              />
            </div>
            <p>Total: {formatBytes(health.disk.total)}</p>
            <p>Used: {formatBytes(health.disk.used)}</p>
            <p>Available: {formatBytes(health.disk.available)}</p>
          </div>
        )}
      </div>
    </div>
  );
}
```
